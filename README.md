# Konstellation network

#### Run singlenet
Run in shell from project dir
```shell script
./scripts/singlenet.sh
```

#### Connect to network
```shell script
konstellation unsafe-reset-all
konstellation config set moniker {MONIKER}
konstellation start
```

#### Run full node
```shell script
./scripts/fullnode.sh
```
